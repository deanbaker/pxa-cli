
# PXA CLI

Go library to check your PXA dollars

## Installation

```bash
go build -o pxa main.go
cp pxa /usr/local/bin
```


## Usage

```bash
> pxa
PXA Summary:
	Shares held: 	291
	Bought at: 	17.13
	Current price:	17.650002
	Current Value:	5136.1504 -> 3.02%
```
